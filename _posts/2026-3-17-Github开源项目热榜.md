# 🔥 二、本周爆发项目深度解读

## 1. OpenViking（字节跳动）

* ⭐ 12.8k ｜本周 +6.5k
* 🔗 [https://github.com/volcengine/OpenViking](https://github.com/volcengine/OpenViking)

专为 AI Agent 设计的上下文数据库。

核心特点：

* 统一管理 memory / resources / skills
* 文件系统范式组织 Agent 上下文
* 支持按需分层加载
* 可自进化

📌 官方定位：为 OpenClaw 等 Agent 提供底层能力

---

## 2. Hermes Agent（NousResearch）

* ⭐ 7.6k ｜本周 +5.1k
* 🔗 [https://github.com/NousResearch/hermes-agent](https://github.com/NousResearch/hermes-agent)

会“自我进化”的 AI Agent。

核心机制：

* 完成任务 → 自动生成 Skill
* 下次任务 → 调用并优化 Skill

特点：

* 越用越强
* Skill 持续积累
* 支持 Telegram / Discord / CLI 等多平台
* 支持多模型一键切换

---

## 3. BitNet（Microsoft）

* ⭐ 34.7k ｜本周 +6k
* 🔗 [https://github.com/microsoft/BitNet](https://github.com/microsoft/BitNet)

1-bit LLM 推理框架。

性能表现：

* ARM CPU：1.4–5x 加速
* x86 CPU：2.4–6.2x 加速
* 能耗降低：55–82%
* 100B 模型可在单 CPU 运行

📌 本地大模型进入全民时代
